# Python 

Python é uma línguagem de programção generalista criada por **Guido Van Rossum** , 1989 através do Instituto de Pesquisa Nacional para Matemática e Ciência da Computação, Guido Van Rossum publicou a primeira versão do Python derivada do C. 

O Python foi criada para que fosse mais simples e produtiva que a línguagem C, e, em 1991 a linguagem Python ganhava sua **versão estável** e já funcional, começando a gerar também uma comunidade dedicada a aprimorá-la.

Somente em 1994 foi lançada sua versão 1.0, ou seja, a sua primeira versão oficial e não mais de testes, e de lá para cá houveram gigantescas melhorias na línguagem em todos os aspectos.

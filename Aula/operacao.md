# Operações aritméticas

Operações são ações ou ações matemáticas e lógicas realizadas sobre dados. Em programação, elas podem ser usadas para manipular valores e variáveis. Existem vários tipos de operações, incluindo:

1. **Operações Aritméticas**: Envolvem cálculos matemáticos básicos.
   - **Adição (`+`)**
   - **Subtração (`-`)**
   - **Multiplicação (`*`)**
   - **Divisão (`/`)**
   - **Divisão inteira (`//`)**
   - **Módulo (`%`)**: Retorna o resto da divisão.

   Exemplo em Python:
   ```python
   a = 10
   b = 3
   soma = a + b  # 13
   resto = a % b  # 1
   ```

2. **Operações de Comparação**: Usadas para comparar valores, retornando `True` ou `False`.
   - **Igual a (`==`)**
   - **Diferente de (`!=`)**
   - **Maior que (`>`)**
   - **Menor que (`<`)**
   - **Maior ou igual a (`>=`)**
   - **Menor ou igual a (`<=`)**

3. **Operações Lógicas**: Usadas para combinar expressões booleanas.
   - **E (`and`)**
   - **Ou (`or`)**
   - **Não (`not`)**

4. **Operações de Atribuição**: Usadas para atribuir valores a variáveis.
   - **Atribuição simples (`=`)**
   - **Atribuição com operação (`+=`, `-=`, etc.)**: Por exemplo, `a += 2` é o mesmo que `a = a + 2`.

5. **Ordem de Precedençia**: Regras de operação.
    1. `{}`
    2. `**`
    3. `*; / ; // ; % `
    4. `+ ; - `

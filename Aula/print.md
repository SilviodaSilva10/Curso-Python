# Print

É umas das funções principais do Python ela permite que usuario possa interagir com o programa exibindo informação para a sua tela.

### print()

```
print('Olá, mundo!')
```

Exibição de atributos de variaveis :

```
nome= 'Silvio'
print('O nome é: {nome}')
```
outra forma:
`print('O nome é {}'.format(nome))`

### Formatação te
```
    nome = 'Silvio'
    print('Ben vindo {:^20}')
```
